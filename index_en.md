This is a one-stop warehouse for collecting, sorting and editing information flow, helping you extract valuable content from the information flow, sort it out, precipitate it, and then output it, so as to build your own knowledge system!

### I Collection information flow type

1. text
2. Markdown
3. web page
Support to save knowledge content as TXT, markdown, pictures and PDF files, support to save web content offline, support to extract wechat official account, Zhihu and today's headline article text as MD document.

### II Collection method

1. share from other apps to apps
2. suspension extract small window
3. enable quick copy creation in knowledge base app settings
4. import browser bookmarks
5. import local file (.Txt.Md.Mht)

### III Classification and editing

1. support classification + label flattening classification management
2. support the three-dimensional construction of knowledge system in tree view
3. provide text, markdown and link editing
4. provide auxiliary functions such as knowledge fragments, full-text search, web page notes, search in web pages, etc
5. support gitee and GitHub drawing beds and two-way links

### IV Backup and recovery

1. local backup
2. WebDAV cloud backup
3. one click export

###  Preferences

Security cabinet, night mode, theme skin changing, font switching, application fingerprint encryption, gesture password and other functions can better improve your use experience!
